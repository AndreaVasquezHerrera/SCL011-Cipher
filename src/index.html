Instrucciones

Para transformar a Mensaje Secreto


Paso 1 : Escribe tu mensaje

Paso 2 : Elige un Idioma Secreto

Paso 3 : Haz "Click" en el boton "cifrar"

Paso 4 :Tu mensaje secreto esta listo!. Con el boton derecho del Mouse puedes copiar tu mensaje para compartirlo con tus amigas!


Para Leer tu Mensaje Secreto

Paso 1 : Escribe tu mensaje (puedes copiar y pegar, con el boton derecho del Mouse)

Paso 2 : Elige el Idioma Secreto con el que se transformo

Paso 3 : Haz "Click" en el boton "descifrar"

Paso 4 :Tu mensaje secreto esta descifrado!.
